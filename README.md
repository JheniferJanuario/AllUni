# AllUni
Esse é um projeto que visa agir como um "wrapper" de anotações para aplicações moodle.

# Comandos


## Migrations
``bash
python manage.py makemigrations
```
``bash
python manage.py migrate
```


## Rodar testes
``bash
coverage run manage.py test
```
``bash
coverage html
```

## Rodar a aplicação
``bash
python manage.py runserver
```
